    # A TO-DO-LIST
 This is a to-do-list program, which can be run in local and deliver necessary functions of a to-do-list.
 
## Install and run the program
 1. Install the programming language Python.
 2. Install the directory **_source_**.
 3. Open the directory **_source_** in terminal.
 4. Run the program by entering the command `py main.py` in terminal.

## How to use the program.
 The program is used via command in terminal.

 - :q : quit the program.
 - :open **_fileName_**: open the **_fileName_** file.
 - :close : save and close the current file.
 - :deleteFile **_fileName_**: delete the **_fileName_** file.
 - :setFileName **_fileName_**: set the name for the current to-do-list.
 - :add **_taskName_**: add the **_taskName_** to the to-do-list.
 - :rename **_index__** **_taskName_**: rename the task at **_index_** to **_taskName_**.
 - :tick **_index_**: tick off the task at **_index_**. Re-enter the command to retick the task.
 - :remove **_index_**: remove the task at **_index_**.
